Dexter - DEMO




We introduce Dexter, an open source framework for entity linking. The entity linking task aims at identifying all the small text fragments in a document referring to an entity contained in a given knowledge base, e.g., Wikipedia. The annotation is usually organized in three tasks. Given an input document the first task consists in discovering the fragments that could refer to an entity (aka spots). Since a spot could refer to multiple entities, it is necessary to perform a disambiguation step, where the correct entity is selected among the candidates. Finally, discovered entities are ranked by some measure of relevance. In this demo you can try to annotate documents, please use the buttons 
dexterize!
to annotate and 
reset!
to insert new documents to annotate. If you move the mouse over a spot (highlighted in red) you will see detailed information on the linked entity.
